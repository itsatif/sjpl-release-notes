Release Notes - Version 1.1.0 - Custom Zoom Library Integration

Features:

  • Integrated the custom zoom library for enabling zoom and pan functionality on SVG elements.
  • Implemented zooming using mouse wheel scroll, touch gestures, and keyboard shortcuts.
  • Supported dragging of the SVG element to pan the view.
  • Added support for resetting the zoom and pan settings using keyboard shortcuts.

Bug Fixes:

  • Fixed an issue where zooming was not limited to the specified range of 0.5 to 3.
  • Resolved a problem where the cursor style was not updated correctly during dragging.
  • Fixed the asset clickability issue in mobile view.
  • Improved responsiveness of zooming for better user experience.

Improvements:

  • Optimized the zooming and panning logic for smoother interaction.
  • Added throttling to mouse dragging events for improved performance.
  • Improved touch event handling for better responsiveness and accuracy.
  • Enhanced keyboard event handling for easier zooming and resetting.

Keyboard Functionality:

  • Press the "+" or "=" key to zoom in by 0.1 increments.
  • Press the "-" key to zoom out by 0.1 increments.
  • Press the "Escape", "r", or "R" key to reset the zoom and pan settings to their initial values.
  • Press the spacebar key to reset the zoom and pan settings to their initial values (while preventing page scrolling).

Note: The custom zoom library provides powerful zooming and panning capabilities for SVG elements, enhancing the interactive experience for exploring and navigating SVG content.
